Live transcription uses the Web Speech API built into modern browsers. It processes your voice in real-time using AI algorithms directly in your browser, without sending data to any server. This ensures 100% privacy while providing accurate transcription.
Live transcription is supported in Google Chrome, Microsoft Edge, and Safari. Firefox has limited support. For best results, we recommend using the latest version of Chrome or Edge.
The Web Speech API typically achieves 90-95% accuracy for clear speech. Accuracy depends on audio quality, background noise, accents, and speaking speed. For best results, speak clearly in a quiet environment.
